# 🚀 Automated Social Media Content Generation using AI (n8n)

## 📌 Overview

This project is an AI-powered automation system that generates and publishes LinkedIn posts from news articles. It uses workflow automation with n8n and integrates multiple APIs to create content, generate images, and post automatically.

---

## ⚙️ Features

* Automatically fetches news article links from Google Sheets
* Summarizes articles using Google Gemini AI
* Generates human-like LinkedIn posts
* Creates AI-based image prompts
* Generates images using Freepik API
* Automatically publishes posts on LinkedIn
* Fully automated end-to-end workflow

---

## 🛠️ Tech Stack

* n8n (Workflow Automation)
* Google Gemini API (AI Content Generation)
* Freepik API (AI Image Generation)
* Google Sheets API
* REST APIs

---

## 🔄 Workflow

1. Google Sheets Trigger fetches article links
2. AI summarizes the article content
3. Generates a LinkedIn post in a human-like format
4. Creates an image prompt using AI
5. Generates image using Freepik API
6. Posts content + image directly to LinkedIn

---

## 🎯 Use Case

This project helps automate content creation for social media, saving time and effort while maintaining consistent posting. It is useful for creators, marketers, and professionals who want to share insights regularly.
